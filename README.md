# Pomodoro Timer
A simple Pomodoro to help with your focus and productivity!

I got this idea from Lofi Girl's Pomodoro timer stream. But with this, you can play any song and use it as there are sound effects to tell it is working time or break time.

There is also another on Lofi Girl's stream whch is that, after 2 cycles(2 work timers and 2 break timers), you get a 15 minute break. I have not added that to this but it is still good in its own way and I don't really bother about it. It really depends if you want it or not. Hope you enjoy it!
